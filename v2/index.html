About

sshPilot is a user-friendly, lightweight SSH connection manager for Linux with an integrated terminal. It's a free alternative to Putty and Termius, designed specifically for the Linux ecosystem.

Manage all your servers in one place with an intuitive interface and powerful features.

Features

  • Tabbed Interface - Manage multiple SSH connections with tabs
  • Keyboard Navigation - Full keyboard support with shortcuts
  • File Management - Integrated SFTP support
  • Secure Storage - Secure credential storage
  • Port Forwarding - Local, Remote and Dynamic port forwarding
  • Key Management - Generate keypairs and ssh-copy-id support
  • Customizable UI - Light/Dark themes and terminal customization
  • Performance - Lightweight and fast with GTK4
